# homebrew-infimount

Homebrew tap for [Infimount](https://github.com/infimount/infimount).

## Install

```bash
brew tap infimount/infimount
brew install infimount
```

## Upgrade

```bash
brew update
brew upgrade infimount
```

## Uninstall

```bash
brew uninstall infimount
```

## Notes

- This tap currently installs the Linux AppImage build through Homebrew (works on Linuxbrew/Homebrew on Linux).
- Formula versions are updated from GitHub releases.
- Security: pin the formula `sha256` from release checksums before publishing updates.
